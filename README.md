# 古文用言 神経衰弱 (kobun-yogen2)

Googleスプレッドシートをデータベースとして活用し、古文の単語（用言）とその「活用の種類」（ラ行変格活用やカ行四段活用など）を神経衰弱（Memory Game）形式で楽しみながら学ぶことができる教育Webアプリケーションです。

1人での学習（独習・タイムアタック）はもちろん、2人対戦でカードを取り合う雅やかな合戦を楽しむこともできます。

---

## 🌸 主な機能

* **単語 ⇔ 活用の種類 マッチングゲーム**
  * 古文の用言（動詞・形容詞・形容動詞）の単語カードと、その活用の種類（「ラ行変格活用」「カ行四段活用」など）のカードをめくって合わせることで、古典文法の基礎知識を定着させます。
* **プレイ人数モード**
  * **一人（独習）**: 手数とクリアタイムを計測し、自身の限界に挑戦するモード
  * **二人（対戦）**: 交互にカードをめくり、獲得したペア数を競い合う対戦モード（ペアを当てると手番が継続します）
* **本日のふり返り（獲得用言一覧）**
  * ゲーム終了後、獲得した用言の基本情報や主要な活用形を一覧表で確認でき、復習・学習効果を高めます。
* **雅やかな和風モダンデザイン**
  * 深紅と金を基調とした和の美しさと、滑らかな3Dカードフリップアニメーションが融合したプレミアムなUI。
* **Googleスプレッドシート連携（GAS）**
  * スプレッドシートを編集するだけで、リアルタイムに出題データを変更・追加できます。GAS接続がない場合でも、内蔵のデモデータで即座に遊ぶことができます。
* **自作データ（CSV/JSON）の直接アップロード & テンプレート保存**
  * スプレッドシート連携のセットアップなしでも、ブラウザ上でCSV（またはJSON）ファイルを直接読み込んでカスタム出題データをロードしてプレイできます。
  * 設定画面から、Excelで開いても文字化けしない「BOM付きUTF-8」形式のCSV入力用テンプレートを1クリックでダウンロード可能です。

## 🛠️ 環境構築・デプロイ手順

このアプリは、フロントエンドを **GitHub Pages**、バックエンドを **Google Apps Script (GAS)** で構築し、完全に無料でデプロイできます。

### 1. Googleスプレッドシート（データベース）の作成

1. 新しいGoogleスプレッドシートを作成します。
   * **⚠️ 重要なセキュリティ上の注意**: このスプレッドシートファイルには、成績表や個人情報などの無関係なデータを含むシート（タブ）を決して同居させないでください。安全のため、完全に独立したゲーム専用のファイルとして作成します。
   * ゲーム用のシート（タブ）名はデフォルトの **`シート1`** としてください。
2. 1行目に以下のヘッダー（カラム名）を入力します。

| A | B | C | D | E | F | G | H | I | J | K |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **ID** | **単語** | **意味** | **品詞** | **活用の種類** | **未然形** | **連用形** | **終止形** | **連体形** | **已然形** | **命令形** |

*※ `ID` は半角数値、その他はテキストで入力してください。*

#### データの入力例
| ID | 単語 | 意味 | 品詞 | 活用の種類 | 未然形 | 連用形 | 終止形 | 連体形 | 已然形 | 命令形 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 1 | あり | 存在する、ある | 動詞 | ラ行変格活用 | あら | あり | あり | ある | あれ | あれ |
| 2 | おどろく | 気づく、目を覚ます | 動詞 | カ行四段活用 | おどろか | おどろき | おどろく | おどろく | おどろけ | おどろけ |
| 3 | うつくし | かわいらしい、いとしい | 形容詞 | シク活用 | うつくしか | うつくしく | うつくし | うつくしき | うつくしけれ | ○ |

### 2. GAS (Google Apps Script) の設定とデプロイ

1. 作成したスプレッドシートのメニューから **「拡張機能」 ＞ 「Apps Script」** を開きます。
2. エディタに記述されている初期コードを全て削除し、本リポジトリに含まれる [gas_code.js](file:///Users/hamadakazuyuki/Documents/古文_用言/gas_code.js) のコードをそのまま貼り付けます。
3. プロジェクトを保存します（`Ctrl + S` または `Cmd + S`）。
4. 画面右上の **「デプロイ」 ＞ 「新しいデプロイ」** をクリックします。
5. 歯車アイコン（種類の選択）から **「ウェブアプリ」** を選択します。
6. 設定を以下のように指定します：
   * **次のユーザーとして実行**: `自分 (YOUR_EMAIL@gmail.com)`
   * **アクセスできるユーザー**: `全員 (Anyone)`
7. **「デプロイ」** ボタンをクリックします。
8. 承認を求められた場合は、「アクセスを承認」をクリックし、ご自身のアカウントを選択して進めます（「詳細」＞「（安全ではないページ）に移動」をクリックして許可します）。
9. デプロイ完了画面に表示される **「ウェブアプリのURL」** をコピーします。
   *(例: `https://script.google.com/macros/s/XXXXX/exec`)*

### 3. フロントエンド（アプリ）の設定と確認

1. アプリをブラウザで開きます。
2. 設定画面の「スプレッドシート API URL」の入力欄に、コピーした **「ウェブアプリのURL」** を貼り付けます。
3. 接続に成功すると、「接続成功: 〇〇件の古文用言データをロードしました」とバッジが表示されます。
4. ※毎回貼り付ける手間を省きたい場合は、[app.js](file:///Users/hamadakazuyuki/Documents/古文_用言/app.js) 内の `const DEMO_WORDS = [...]` の手前に、デフォルトのGAS URLを設定するなどのカスタマイズも可能です。入力されたURLはブラウザ（ローカルストレージ）に自動保存されます。

### 4. GitHub Pages へのデプロイ（Web公開）

作成したコードをGitHubにプッシュし、世界中に公開します。

1. ローカルでGitリポジトリを初期化し、GitHubにプッシュします。
   ```bash
   git init
   git add .
   git commit -m "Initial commit of kobun-yogen2 game"
   git branch -M main
   git remote add origin https://github.com/Hamakazu0219/kobun-yogen2.git
   git push -u origin main
   ```
2. GitHubのリポジトリページ（`https://github.com/Hamakazu0219/kobun-yogen2`）を開きます。
3. **「Settings」** タブをクリックします。
4. 左メニューの **「Pages」** をクリックします。
5. **「Build and deployment」** の **「Source」** を `Deploy from a branch` に設定します。
6. **「Branch」** で `main` と `/ (root)` を選択し、**「Save」** をクリックします。
7. 数分後、ページ上部に `Your site is live at https://hamakazu0219.github.io/kobun-yogen2/` のように公開URLが表示され、アクセス可能になります。

### 5. Vercel へのデプロイ

より素早く簡単に公開したい場合は、Vercel を使用してワンコマンドでデプロイ可能です。

1. Vercel CLI をインストール・ログインします：
   ```bash
   npm install -g vercel
   vercel login
   ```
2. プロジェクトのルートディレクトリでデプロイを実行します：
   ```bash
   vercel --prod
   ```
   *※ プロジェクトのルートフォルダ名に日本語などの非ASCII文字が含まれている場合、CLIの自動紐付けエラーを引き起こす可能性があるため、フォルダ名を半角英数（例: `kobun-yogen2`）にしてから実行することをお勧めします。*

## 📄 ファイル構成

* **`index.html`**: ゲーム全体の画面構造（設定、ゲーム画面、結果モーダル）を記述したHTML。
* **`style.css`**: 和モダンなプレミアムデザイン、レスポンシブグリッド、3Dフリップアニメーションを定義したCSS。
* **`app.js`**: ゲームの状態管理、シャッフル、マッチング判定、GASデータフェッチ、振り返り処理を実装したJavaScript。
* **`gas_code.js`**: スプレッドシートからデータを取得しJSON出力する、GAS用の参照コード。
