# メモリーマッチ！カスタム暗記カード (Memory Match! Custom Learning Cards)

GoogleスプレッドシートやCSVファイルをデータベースにして、あらゆる暗記カードゲームを自分で作って遊べる、汎用的な教育Webアプリケーションです。

1人での学習（タイムアタック）はもちろん、2人プレイでカードを取り合う対戦モードも搭載した、ポップでモダンなグラスモルフィズムデザインのゲームです。

---

## ⚡ 主な特徴

* **組み合わせ自由自在（動的カラム検出）**
  * アップロードしたCSVやスプレッドシートの列名（ヘッダー）を自動で解析し、ゲームの「A面」と「B面」に合わせたい項目をボタンで自由に選択できます。
  * 例：「単語 ⇔ 意味」「国名 ⇔ 首都」「歴史の出来事 ⇔ 年号」「星座 ⇔ 明るい星」など、何でも組み合わせ可能です。
* **プレイ人数モード**
  * **1人プレイ**: 手数とクリアタイムを計測し、自分の限界に挑戦するタイムアタック学習モード。
  * **2人プレイ**: 交互にめくり、獲得したペア数を競い合う対戦モード（ペアが揃うと続けてめくれます）。
* **学習のまとめ・ふり返り一覧**
  * ゲーム終了後、獲得したカードのデータが表形式で一覧表示され、間違えた項目や覚えたい情報をしっかりと復習できます。
* **ポップ＆モダンなネオンデザイン**
  * 深みのあるインディゴと鮮やかなネオンピンク・シアンのグラデーション、そして半透明のグラスモルフィズムスタイルを取り入れた、最先端で高級感のあるUI。
* **Googleスプレッドシート連携（GAS）**
  * スプレッドシートを更新するだけで、クラウド経由でリアルタイムに出題データを変更・追加できます。
* **自作ファイル（CSV/JSON）の直接アップロード**
  * スプレッドシートの設定が面倒な場合でも、ブラウザ上でCSV（またはJSON）ファイルを直接ドラッグ＆ドロップ（選択）してロードするだけで即プレイ可能です。 Excelで開いても文字化けしない「BOM付きUTF-8」形式の入力テンプレートも1クリックで保存できます。

## 🛠️ 環境構築・デプロイ手順

### 1. Googleスプレッドシート（データベース）の作成

1. 新しいGoogleスプレッドシートを作成します。
   * **⚠️ 注意**: セキュリティのため、個人情報や成績情報などの無関係なデータが入っているシートとはファイルを分けて作成してください。
   * ゲーム用のシート（タブ）名は **`シート1`** としてください。
2. 1行目に好きなカラム（列）名を入力し、2行目以降にデータを記述します。

#### 入力例（英単語）
| A | B | C | D | E |
| :--- | :--- | :--- | :--- | :--- |
| **ID** | **単語** | **意味** | **品詞** | **分類** |
| 1 | apple | りんご | 名詞 | フルーツ |
| 2 | run | 走る | 動詞 | 動作 |
| 3 | beautiful | 美しい | 形容詞 | 状態 |

#### 入力例（地理・社会科）
| A | B | C | D |
| :--- | :--- | :--- | :--- |
| **ID** | **都道府県** | **県庁所在地** | **地方名** |
| 1 | 北海道 | 札幌市 | 北海道 |
| 2 | 青森県 | 青森市 | 東北 |

*※ `ID` は半角数値、その他はテキストで入力してください。*

---

### 2. GAS (Google Apps Script) の設定

1. 作成したスプレッドシートのメニューから **「拡張機能」 ＞ 「Apps Script」** を開きます。
2. 初期コードを全て削除し、本リポジトリの [gas_code.js](file:///Users/hamadakazuyuki/Documents/古文_用言/gas_code.js) のコードをそのまま貼り付けます。
3. 保存し、右上の **「デプロイ」 ＞ 「新しいデプロイ」** をクリックします。
4. 設定を以下のように指定します：
   * **種類の選択**: `ウェブアプリ`
   * **次のユーザーとして実行**: `自分 (YOUR_EMAIL@gmail.com)`
   * **アクセスできるユーザー**: `全員 (Anyone)`
5. デプロイボタンをクリックし、表示される **「ウェブアプリのURL」** をコピーします。
   *(例: `https://script.google.com/macros/s/XXXXX/exec`)*

---

### 3. フロントエンド（アプリ）の設定とWeb公開

1. アプリをブラウザで開き、設定画面の「スプレッドシート API URL」にURLを貼り付けます。
2. 接続に成功すると、「接続成功: 〇〇件のデータをロードしました」とバッジが表示されます。
3. **GitHub Pages への公開**:
   本リポジトリをGitHubにプッシュし、リポジトリの「Settings」＞「Pages」から `Deploy from a branch` で `main` ブランチを指定するだけで無料公開されます。
4. **Vercel へのデプロイ**:
   Vercel CLIを使用してコマンドラインから一瞬で公開できます。
   ```bash
   npm install -g vercel
   vercel --prod
   ```

## 📄 ファイル構成

* **`index.html`**: アプリケーションの画面構造（設定画面、ゲーム画面、結果表示モーダル）。
* **`style.css`**: ポップでモダンなネオンのグラスモルフィズムデザイン、3Dカードアニメーション。
* **`app.js`**: データフェッチ、動的ヘッダー解析、ゲームステートの管理、マッチ判定、復習テーブルの描画。
* **`gas_code.js`**: Googleスプレッドシートからデータを安全にJSON出力するGAS用の参照コード。
